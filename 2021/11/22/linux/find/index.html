find

find简介

文件查找工具

find使用

find 文件路径 参数

例子:

find ~ -name 'test'

mac系统有个权限问题:

find: /Users/xx/.Trash: Operation not permitted

mac的安全机制，需要重启电脑，按下 Command + R进入恢复模式，关掉安全机制。但是正常来说还是别这么干，不划算。

csrutil disable

locate命令

locate 与 find 不同: find 是去硬盘找，locate需要建立一个数据库/var/lib/slocate/slocate.db。

locate test #查找test开头的文件
locate /Users/test  #/Users下查找test开头的文件
